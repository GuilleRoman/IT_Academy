M6.3 exercici 1

Enunciat:

Fer un programa on l’usuari introdueix per pantalla un caràcter i un número. El programa imprimeix per pantalla un quadrat amb la mida i el caràcter introduïts per l’usuari.
Exemple:
% % % % %
% % % % %
% % % % %
% % % % %
% % % % %
